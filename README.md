# ai-boston-housing
AI Boston Housing

### Setup
1. Install Miniconda - https://docs.conda.io/en/latest/miniconda.html
2. conda env remove -n ds_base_env
3. conda env create -f ds_base_env.yaml
4. conda info --envs
5. conda env list
6. conda activate ds_base_env
7. jupyter notebook
8. conda deactivate
